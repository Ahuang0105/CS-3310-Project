CS 3310 Project1

The IDE used to build this programs is Eclipse, the zip file "AlanHuang_project1.zip" file
contain 1 folder call "CS3310 Project 1". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS3310 Project 1" folder and clip "finish". Also, the source
code is locate in the folder call "src" .


The file main.java has the main method that will test all 4 of the task for this project. 
After running the main method it will give you 4 options to run such as mergesort, quicksort,
tower of Hanoi, or matrix multiplication. If you want the exit the program just simply enter 0.
