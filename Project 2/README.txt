CS 3310 Project2

The IDE used to build this programs is Eclipse, the zip file "AlanHuang_project2.zip" file
contain 1 folder call "CS3310 Project 2". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS3310 Project 2" folder and clip "finish". Also, the source
code is locate in the folder call "src" .


The file project2.java has the main method that will test all bost of the task for this project. 
After running the main method it will ask you to input capacity, how many object, and each object's
weight and profits. After that the program will return the results.

For executable file please use Command prompt to run the .jar file. First you will need to change the 
directory to where the file is, then use "java -jar AHP2.jar" to run the program. For example if you 
place the AHP2.jar at the Desktop. You will need to enter "cd Desktop" then enter "java -jar AHP2.jar".
" " is use the highlight the command.
